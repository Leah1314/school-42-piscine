# School 42 Piscine

This repository contains coursework and group project work from the one-month intensive 42 Piscine at School 42 London, completed in 2025.

I successfully passed the Piscine at the end of the programme.

## About The Piscine

The 42 Piscine is an intensive, peer-to-peer coding programme focused on learning through practical exercises, collaboration, and problem solving. The coursework in this repository is primarily written in C and covers core programming concepts such as:

- Basic syntax and program structure
- Functions and recursion
- Pointers and memory management
- String manipulation
- Arrays and command-line arguments
- Algorithms and problem solving
- Unix shell basics
- Collaborative group projects

## Repository Structure

- `c00_v2` to `c07_v2` - C coursework exercises completed during the Piscine
- `c01`, `c03`, `c04`, `c05`, `c06`, `c07` - additional or earlier exercise versions
- `rush00`, `rush01` - group project work
- `mock` - mock exam practice and related materials
- `Others` and `test` - additional practice files and experiments
- `*.subject.pdf` - subject PDFs and exercise briefs
- `en.norm.pdf` - 42 coding norm reference

## Skills Practised

- Writing C programs from scratch
- Debugging with compiler feedback and manual testing
- Working with pointers, strings, loops, and recursion
- Breaking problems down into smaller functions
- Reading and following technical specifications
- Collaborating in a peer-learning environment
- Managing time and pressure during an intensive coding programme
