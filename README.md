# School 42 Piscine Coursework

This repository contains coursework, practice exercises, and group project work from the one-month intensive 42 Piscine at School 42 London, completed in 2025.

I successfully passed the Piscine at the end of the programme.

## Overview

The 42 Piscine is a highly intensive selection and learning programme built around peer-to-peer education, self-directed study, practical problem solving, and daily coding challenges. During the Piscine, I worked primarily in C while learning how to approach unfamiliar problems, debug independently, collaborate with peers, and complete work under time pressure.

This repository is a record of that learning process. It includes submitted coursework, practice attempts, subject PDFs, mock exam preparation, and group project folders.

## About The Piscine

The coursework in this repository is primarily written in C and covers core programming concepts such as:

- Basic syntax and program structure
- Functions and recursion
- Pointers and memory management
- String manipulation
- Arrays and command-line arguments
- Algorithms and problem solving
- Unix shell basics
- Collaborative group projects
